#### Installation and Usage:

Copy-paste this into your terminal:

```
git clone https://github.com/CoderSigma/SiteCrawler.git
```
```
cd SiteCrawler
```
```
pip install -r requirements.txt

```
```
python3 python3 scanner.py
```

#### Usage
```
usage: python3 scanner.py

Choose an option:
1. Normal Scan (Uses wordlist.txt)
2. Deep Scan (Crawls all accessible pages, images, and files)
3. Hybrid Scan (Combines both scans)
Enter your choice (1, 2, or 3): 
```
